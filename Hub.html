AXIS Hub is a routing screen only. It does not create quotes, save rep actions, save manager responses, or make operational decisions. It packages records by manager and writes routed queues into local browser storage for the Manager, Rep, and VP views.

AXIS outputs are informational planning signals only. Final decisions remain with the OEM, customer, and their authorized stakeholders.
